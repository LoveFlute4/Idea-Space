DIY Relay-Based Sequential Timer

What it is

A project to create a mechanical timer using relays that turn on/off in a specific sequence.

Basic principle

Uses electromagnetic relays arranged in a way to pass current in timed stages. Each relay triggers the next after a delay.

Type of project

Full Hardware

How I think I'd start

Start with a 12V power supply, SPDT relays, and a basic understanding of how to create time delays using RC circuits or thermal delay. Use LEDs as indicators for each stage and test sequencing manually before automating.
