DIY Magnetic Levitation Device

What it is

A setup that levitates a small object (like a magnet or metal ball) using electromagnetic repulsion or attraction.

Basic principle

Electromagnets controlled by a feedback system using Hall sensors or IR sensors to detect position and stabilize the object mid-air.

Type of project

More Hardware Than Software

How I think I'd start

Begin with a powerful electromagnet, a steel or magnetic object, and a Hall effect sensor. Control the magnet's strength using a transistor and adjust based on sensor feedback using an Arduino Nano.
